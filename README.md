# Colección Beatriz González

Colección de obras y referentes de la artista Beatriz González.

## Contenidos

En la carpeta ```data``` se encuentran los metadatos de las obras.  Por un lado se incluye una tabla en formato ```.xlsx``` con dos hojas, una para obra y otra para los referentes.  Igualmente se incluye cada una de estas tablas en formato ```.csv```. Los campos de los metadatos han sido normalizados con el estándar Dublin Core

El la carpeta ```img``` se encuentran los archivos correspondientes a las obras y referentes.
